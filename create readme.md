# AI API Integration Project

## Description
This project demonstrates integration of Google Gemini API using Python.

## Features
- Accepts user input
- Sends request to AI API
- Displays response
- Handles errors

## Technologies Used
- Python
- Google Gemini API

## How to Run
1. Install requirements:
   pip install -r requirements.txt
2. Run:
   python main.py
   Multiple APIs can be integrated similarly (OpenAI, HuggingFace).
